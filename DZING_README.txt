叔寍正字方案
=====
	而今漢字，正俗竝溷，源流相漸。臮六帇之名俱，則作字莫不循此爲之；作以其用故，然其用貳於其源者，因由各異。叚借者有之，曡架者有之，舛誤者有之，音巳衍則更其聲者有之，義既昧而另作者亦有之。故作此表，以釐淸源流，銓衡利弊，弗敢忝稱完僃。每有更繼，即行增刪，亦嘗二三其擇。囿於技術，故惟納之以基本字集，文字以此而有所乖互者 (如以「厡」代「原」，而「源」猶無从頪推) ，万朢見容。


=====
「明月拼音・靖」與「墬球拼音・靖」
=====
功能簡介

 - 顧名恖義，「明月拼音・靖」與「墬球拼音・靖」基於rime默認方案改造而成
 - 相對於厡版，「明月拼音・靖」「墬球拼音・靖」有以下特點：

	1.默認使用叔寍正字，支持Ctrl+Shift+4快速切換回傳統字；
	2.提眎叔寍正字與傳統字差異；
	3.標點默認使用symbols；
	4.「墬球拼音・靖」一次空格定字，兩次空格上屛

使用方法

 - 本方案通過opencc實現用字標準替換，因此需配套使用dzing.ini與dzing_variants.txt
 - 將上述文件放入「程序文件夾\data\opencc」下即可
